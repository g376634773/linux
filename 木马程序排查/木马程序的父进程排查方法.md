马程序的父进程排查方法
=====================
生成木马的父进程，自动检测子进程，如果子进程被删除，父进程可以自动启动子进程。  这就是Linux删除木马后，木马自动又生成。
1、首先先排查到木马程序进程  
2、然后查找父进程  
```
# pstree  | grep freg  
|-gnome-terminal-+-bash---workstat---2*[fregonnzkq---sleep]
```  
3、查看父进程路径  
```
# ps -axu | grep workstat   
root      7533  2.1  0.1 106152  1288 pts/0    S    10:29   0:12 /bin/bash /bin/workstat
```  
4、可以查看一下父进程内容  
``` # vim /bin/workstat ```  
5、删除父进程  
``` # rm -rf /bin/workstat ```  
6、删除父进程中调用的病原体  
```
# rm -rf /usr/wke   
# kill -9 7533
# killall fregonnzkq
```
